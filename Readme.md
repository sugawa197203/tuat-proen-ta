# プロ演TA半自動採点システム

このReadmeちゃんと読んでください、

# 前提

- OS
  - Ubutu 24.04
- ターミナル
  - bash

## 注意事項

- Windows では動かないです。WSLとか仮想環境を使ってください。
- Mac での動作チェックはしてないです。
- 環境とかよくわからないなら、とりあえずWSLでUbuntu 24.04入れてください。

# 環境構築

## 1. 必要なパッケージをインストール

```bash
sudo apt install -y build-essential clang git
```

## 2. 本リポジトリをクローン

```
git clone https://github.com/sugawa197203/proen-ta
```

- 以前にクローンしたことあるならmainをpull

```
git pull origin main
```

環境構築ができたらVSCodeで開きましょう。そのほうが楽だと思います。

# 採点方法

## 1. ディレクトリの移動

問題ごとにディレクトリが分かれているので、採点したい問題のディレクトリに移動してください。

```
cd <ディレクトリ名>
```

- 例

前期第1回の課題

```
cd 1-1
```

## 2. `target.c` に採点対象のコードをコピペ

問題のディレクトリに `target.c` というファイルがあります。これが採点対象のコードを入れるファイルです。ここにコードをコピペしてください。

## 3. `run.sh` を実行

```
./run.sh
```
